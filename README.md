# Trivia Clip Factory

Public website for Trivia Clip Factory, including homepage, Terms of Service, and Privacy Policy pages used for TikTok Developer Portal review.

GitHub Pages deploys the `website/` directory through `.github/workflows/pages.yml`.

After deployment, use these URLs in TikTok Developer Portal:

- Website: `https://VinodGeorge24.github.io/trivia-clip-factory/`
- Terms: `https://VinodGeorge24.github.io/trivia-clip-factory/terms.html`
- Privacy: `https://VinodGeorge24.github.io/trivia-clip-factory/privacy.html`
