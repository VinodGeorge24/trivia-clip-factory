# Trivia Clip Factory Website

Static public pages for TikTok Developer Portal review:

- `index.html`
- `terms.html`
- `privacy.html`

No build step is required. Host the contents of this directory on any static
site host and use the resulting URLs in TikTok Developer Portal.

This repository includes `.github/workflows/pages.yml`, which deploys this
directory through GitHub Pages when Pages is configured to use GitHub Actions.

Recommended URLs after hosting:

- Homepage: `https://YOUR_SITE_URL/`
- Terms of Service: `https://YOUR_SITE_URL/terms.html`
- Privacy Policy: `https://YOUR_SITE_URL/privacy.html`
